2026 KUSF 대학배구 U-리그 규정

공식 대회요강에서 경기 방식, 순위 결정 기준, 본선 진출 방식과 시상 항목을 구조화했습니다.

규정 구조화 완료계산기 데이터 연결 준비 중

대회 개요

  • 기간: 2026년 3월 25일 ~ 10월 6일
  • 장소: 참가 대학교 체육관
  • 방식: 홈&어웨이
  • 주최: 한국대학스포츠협의회
  • 주관: 한국대학배구연맹

남대부 경기 방식

  1. 조별 예선리그
  2. 각 조 상위 3팀 진출
  3. 총 6팀이 6강 리그 진행
  4. 준결승전
  5. 챔피언결정전

승점 기준

  • 3-0·3-1 승리: 승리팀 3점 / 패배팀 0점
  • 3-2 승리: 승리팀 2점 / 패배팀 1점

순위 결정 순서

  1. 승점
  2. 승리경기 수
  3. 득실점수비율
  4. 세트비율
  5. 동률 팀 간 승자승

3개 이상 팀 동률 처리

승점, 승리경기 수, 득실점수비율, 세트비율이 모두 같은 3개 이상 팀이 발생하면 해당 팀 간 경기만 다시 산정하여 승리경기 수 → 득실점수비율 → 세트비율 순으로 결정합니다.
