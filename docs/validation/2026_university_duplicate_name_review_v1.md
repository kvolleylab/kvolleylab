# 2026 대학 남자배구 중복 이름 검토 v1

## 검토 대상

1. 중부대학교 정현우: 등번호 8번 / 18번
2. 조선대학교 김우진: 등번호 9번 / 6번
3. 경상국립대학교 박세진: 등번호 10번 / 1번

## 확인 결과

### 1. 중부대학교 정현우

- 원본 명단에는 등번호 8번과 18번 두 행이 존재한다.
- 공개 경기 중계에서는 정현우 선수를 중부대학교 세터로 소개하며 신장 193cm라고 언급한다.
- 현재 자료만으로는 해당 선수가 8번인지 18번인지, 또는 두 행이 동명이인인지 중복 입력인지 확정할 수 없다.
- 처리: 두 Seed 행 유지, 영구 ID 발급 보류.

### 2. 조선대학교 김우진

- 원본 명단에는 등번호 9번과 6번 두 행이 존재한다.
- 두 행 모두 기본 프로필이 동일하게 MB, 190cm로 반영되어 있다.
- 별도 공개 자료에서 두 명의 동명이인 선수임을 확인하지 못했다.
- 처리: 중복 입력 가능성이 높지만 원본 보존 원칙에 따라 두 Seed 행 유지, 영구 ID 발급 보류.

### 3. 경상국립대학교 박세진

- 원본 명단에는 등번호 10번과 1번 두 행이 존재한다.
- 두 행을 구분할 추가 프로필 정보가 없다.
- 별도 공개 자료에서 두 명의 동명이인 선수임을 확인하지 못했다.
- 처리: 두 Seed 행 유지, 영구 ID 발급 보류.

## 결론

- 3개 이름 묶음 모두 현 단계에서는 확정 병합 또는 삭제하지 않는다.
- 총 6개 Seed 행은 `needs_review` 상태를 유지한다.
- 공식 선수 등록 명단, 학교 공식 로스터, 경기 출전 기록 중 등번호가 확인되는 자료 확보 후 최종 판정한다.
- 나머지 223개 Seed 행은 기존 Player Master 중복 대조 대상으로 이동할 수 있다.
