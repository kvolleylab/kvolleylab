## 검증 기준

1. 공식 대학·연맹·협회 자료를 우선 사용한다.
2. 기사·대회 신입생 현황은 보조 출처로 사용한다.
3. 확인한 모든 값은 출처 URL과 확인일을 함께 저장한다.
4. 생년월일이 확인되기 전에는 `display_code`를 생성하지 않는다.
5. 미확인 값은 추정하지 않고 `null`을 유지한다.
